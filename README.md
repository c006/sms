Yii2  SMS
===================

Send text message (requires sign up to site)


Installation
------------


```
php composer.phar require --prefer-source "c006/yii2-sms" "dev-master"
```

or add

```
"c006/yii2-sms": "dev-master"
```


Usage
-----


>
    $sms = new Sms();
    $sms->send($form_phone_number, $subject, $message);
